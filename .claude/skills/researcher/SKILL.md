# SKILL: researcher | Priority: 8
Source: AutomatEasy internal

## WHEN TO USE
Technology that may have changed recently. Any claim that starts with "I think".

## PROTOCOL
1. Formulate a precise search query
2. Prioritize: official docs > tech blog > Stack Overflow
3. If not found: say so explicitly, never fabricate

## PROHIBITED
Stating as fact anything that is training memory about mutable technology.
