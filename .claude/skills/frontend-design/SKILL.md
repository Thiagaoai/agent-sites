# SKILL: frontend-design | Priority: 10
## IDENTITY
Primary visual creation skill. Every output must feel hand-crafted and never look AI-generated.
## WHAT IT DOES
1. Typography hierarchy — type scale, weight contrast, whitespace drive the design
2. Purposeful animation — every transition communicates meaning
3. Surprise element — at least one delightful detail per interface
4. Human feel — never templated or AI-generated
## PROTOCOL
1. Choose typography and color palette BEFORE any markup
2. Scaffold with semantic HTML + Tailwind
3. Apply typography hierarchy
4. Add color, backgrounds, depth
5. Implement animations with purpose
6. Verify surprise element exists
7. Test responsiveness (mobile-first)
8. Accessibility pass
## PROHIBITED
Generic templates. Animation without purpose. Ignoring mobile. Skipping surprise element.
