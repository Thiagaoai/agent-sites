# SKILL: cost-reducer | Priority: 9
Source: AutomatEasy internal

## WHEN TO USE
Long sessions. Context above 50% of window. Repetitive tasks.

## MANDATORY PRACTICES
- Never include entire file when only a function is needed
- Never repeat already-defined code — reference by name
- Use /compact before starting a new phase in a long session
- Avoid re-fetching data already in context
