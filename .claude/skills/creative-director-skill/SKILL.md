# SKILL: creative-director-skill | Priority: 10
## IDENTITY
Defines creative concept, visual narrative, and emotional direction BEFORE any code is written.
## 5 PHASES
1. Briefing: find real objective, audience, emotion target
2. Divergence: 20+ ideas using SCAMPER, Bisociation, Analogy Transfer
3. Convergence: Originality 30% + Relevance 25% + Feasibility 20% + Emotional 15% + Differentiation 10%
4. Creative Direction: Concept, Emotional Target, Visual Metaphor, Key Decisions, Anti-Patterns, References
5. Gate: "Would any AI produce this without this briefing?" → YES = go back to Phase 2
## PROHIBITED
Jumping to execution without concept. Accepting first idea. Copying competitors.
